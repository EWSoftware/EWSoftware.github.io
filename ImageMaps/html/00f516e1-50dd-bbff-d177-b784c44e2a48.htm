Remarks
Normally, the control will search for the form control on the page when it needs to register the helper script for post back image areas. If it cannot find the form control, it will default to using index zero (forms[0]) instead. If you have other form controls on the page (i.e. normal HTML forms in addition to the ASP.NET Run at Server form), you may need to use the FormId property to specify the actual form ID to use for the post back script. If specified, that form ID will override the search behavior and will always be used. If null or blank, the above noted search and fallback behavior will be used instead.
