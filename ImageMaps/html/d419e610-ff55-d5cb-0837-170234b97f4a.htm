      EWSoftware.ImageMaps Namespace

      This is the main name space for all of the ImageMap control classes. It contains classes common to both the Windows Forms and web server controls.

      Classes

      ImageAreaCollection A type-safe collection of IImageArea objects
      ImageMapClickEventArgs This is a custom event arguments class for the Click event
      PointCollection A type-safe collection of Point objects.

      Interfaces

      IImageArea This is the common interface for both the Windows Forms and the web server control image area types
      IImageMap This interface defines events, properties, and methods common to both the Windows Forms and the web server control image map classes.

      Enumerations

      ImageAreaShape This enumerated type defines the various image area shapes
