The first address with an address type matching one of those specified or null if not found
Remarks
Multiple address types can be specified. Including Preferred will limit the match to the one with the Preferred flag set. If a preferred address with one of the given types cannot be found, it will return the first address matching one of the given types without the Preferred flag set. If no address can be found, it returns null.
