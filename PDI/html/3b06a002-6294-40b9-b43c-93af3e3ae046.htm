      VTODO Classes

      The VToDo class represents a To-Do component in a calendar. This component is used by both vCalendar and iCalendar.

      AttachProperty and AttachPropertyCollection

      FormatType, GetAttachmentBytes, SetAttachmentBytes, and Value
