Version 1.2.0.5

Version 1.2.0.5 was released on October 8th, 2005.

Changes in This Release

  • Added the MergingEnabled property to the VTimeZoneCollection to allow disabling of the merging behavior when adding time zone objects to the collection.

  • Added an overload to VTimeZoneCollection.IndexOf to optimize the Merge method and significantly improve its performance.

  • Added ToLocalTime and ToTimeZone methods to the DateTimeInstance and DateTimeInstanceCollection classes.
