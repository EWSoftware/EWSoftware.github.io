      TriggerProperty Class

      This class is used to represent the trigger (TRIGGER) property of an iCalendar VAlarm object.

      Remarks

      This property class parses the Value property to allow access to its content as an actual DateTime object or Duration object. The property value is a character string conforming to ISO 8601.
