      Remarks

      It implements functionality common to all properties. It also provides ToString methods used to convert the instance to a format suitable for writing to a PDI data stream.

      Constructors

      BaseProperty Constructor. By default, it uses 7-bit ASCII encoding for the value, and the value type or location is determined from the DefaultValueLocation property.
