TimeZoneUrlProperty Class
This class is used to represent the Time Zone Uniform Resource Locator (TZURL) property of a VTIMEZONE object. This URL may be used to obtain real-time or up-to-date information about the time zone object.

Remarks
This property has no special requirements or handling. The Value property contains the URL. It is not validated for correctness but should conform to the IETF RFC 1738, Uniform Resource Locators.
