DayInstance Class

A day instance item. The properties of this class define a day of the week instance on which a recurrence date may fall.

Remarks
The day instance can be generic specifying that all instances are included (i.e. all Mondays) or it can refer to a specific instance (i.e. the second Monday, the third from last Tuesday, etc).
