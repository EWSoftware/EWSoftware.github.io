      Version 2.0.0.2

      Version 2.0.0.2 was released on September 4th, 2007.

      Changes in This Release

      • Fixed some issues in the VCalendar time zone conversion methods that caused incorrect times or time zone names when crossing a standard to daylight saving time boundary and vice versa.
      • Added support for parsing vNote objects defined in the IrMC 1.1 specification (see VNoteParser, VNote, VNoteColllection, and BodyProperty). A new section for vNote was added to the PDI Tag to Class/Property Mappings topic.
