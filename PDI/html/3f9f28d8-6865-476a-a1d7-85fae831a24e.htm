OrganizerProperty Class

This class is used to represent the organizer (ORGANIZER) property of an iCalendar component. This property defines the organizer within a calendar component.

Remarks
The Value property contains the organizer information. There are some additional parameters that may appear on the object. These are parsed and can be accessed via the class properties. This property class is only used by iCalendar 2.0 objects. It is ignored for vCalendar 1.0 objects. There can be only one organizer so it does not have a collection class.
