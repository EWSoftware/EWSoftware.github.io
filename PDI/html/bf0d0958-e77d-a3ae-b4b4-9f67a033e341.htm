The interval between occurrences in days
Remarks
This is a convenience method that mimics the daily recurrence pattern in Microsoft Outlook. When called, it sets up the recurrence for a daily pattern that recurs at the specified interval. All rule parts are cleared prior to setting the daily options but other parameters such as the start date are left alone.
