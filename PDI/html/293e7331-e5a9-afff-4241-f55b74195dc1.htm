Remarks

This adds a standard set of holidays to the collection that the author gets to take. If you do not get the same set, you can modify the collection after the call or derive a class and override this method to provide the set that you need.

As an alternative, you could serialize a set of holidays to an XML file and load them from it instead.

Fixed dates are set to be adjusted to not fall on a weekend. The holidays added are as follows:
