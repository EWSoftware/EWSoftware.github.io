      Welcome

      Welcome to the Extended XML Documentation Comments Completion Provider project.

      This project is a Visual Studio editor extension that augments the standard C# XML documentation comments elements with custom elements, attributes, and attribute values supported by the Sandcastle Help File Builder and other documentation tools. The following elements, attributes, and attribute values are provided:
