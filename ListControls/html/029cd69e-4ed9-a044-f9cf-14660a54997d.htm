MultiColumnComboBoxColumnFilter Property

This gets the StringCollection used to filter the columns displayed by the drop-down portion of the combo box.

Property Value

Type: StringCollection
This is a quick way to filter the drop-down to a specific set of columns without defining column definitions using the DropDownFormat.GridColumnStyles property collection. If empty, no filtering takes place.
