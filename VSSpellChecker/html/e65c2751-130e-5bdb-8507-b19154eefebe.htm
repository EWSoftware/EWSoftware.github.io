VisualStudio.SpellChecker.Definitions Namespace

This namespace contains classes used to create new tagger providers for third-party content types so that they can make use of the spell checker.
Interfaces
  InterfaceDescription
INaturalTextTag
This represents a tag for natural text regions
ISpellingIssue
This represents a spelling issue such as a misspelled word, doubled word, etc.
ISpellingService
This is used to retrieve the spelling service
ISpellingSuggestion
This represents a spelling suggestion that can be used to replace a misspelled word
