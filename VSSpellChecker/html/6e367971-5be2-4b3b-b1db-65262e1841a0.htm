    Version 2024.12.14.0

    Changes made in this release:

    • An updated was required due to a change in the Visual Studio SDK that forced the spell checker assemblies to be marked as not CLS compliant. Also, when running out of process, the hosting process now makes a copy of the code analyzer assembly in a cache location without any of its dependencies. This required coming up with a workaround to let it find them. These changes fix the failure to load in Visual Studio 2022 (17.12 and later).
