      Release notes for version 2022.4.16.0.

      Changes in This Release

      • Made various general revisions to reflect how the MAML elements are currently used.

      • Added a topic for the bibliography/cite elements.

      • Added a topic for the languageKeyword element.

      • Replaced the title topic with a section topic which makes better sense.

      • Updated the alert element topic with information on a new todo alert type and a new optional title attribute that can be used to override the default alert title.

      • Added documentation for the start attribute on the list element for ordered lists.

      • Added documentation for the new contentType attribute on the markup element.
