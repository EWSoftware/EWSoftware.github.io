      Project Properties Window

      All of the main project properties except documentation sources and references can be edited via the Project Properties window. Project properties are grouped into categories ordered by frequency of usage. Each set of properties is edited using standard controls such as text boxes, combo boxes, list boxes, etc. with descriptive labels. Since there is only one website property, it has been combined with the Help 1 properties in the Help 1/Website category. An information and/or warning tip icon appears after some properties. Hover the mouse over the icon to see additional usage notes or warnings related to the property. For full details about the properties in any given category, press the F1 key while a control on the property page has the focus to show the related help topic for the property category.
