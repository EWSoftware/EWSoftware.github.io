The IMemberIdUrlResolver type exposes the following members.

Properties
  NameDescription
CachedUrls
This read-only property returns the URL cache

This read-only property is used to return the number of items added to the cache during the latest run.
DisabledReason
This read-only property returns the reason the resolver is disabled if IsDisabled returns true.
IsDisabled
This read-only property indicates whether or not the resolver is disabled
IsDisposed
This read-only property can be used to determine whether or not the resolver has been disposed
Locale
This is used to get or set the locale for the reference links if applicable to the resolver
