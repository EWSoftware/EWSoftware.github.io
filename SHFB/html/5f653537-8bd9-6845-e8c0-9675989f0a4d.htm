SandcastleBuilder.WPF Namespaces

This namespace contains the WPF related namespaces
Namespaces
NamespaceDescription
SandcastleBuilder.WPF
This namespace contains common classes used by the WPF user controls.
SandcastleBuilder.WPF.Behaviors
This namespace contains behavior classes used by the WPF user controls.
SandcastleBuilder.WPF.Commands
This namespace defines new commands used by the help file builder WPF user controls.
SandcastleBuilder.WPF.Controls
This namespace contains WPF custom controls.
SandcastleBuilder.WPF.Maml
This namespace contains various classes used to convert MAML topics to WPF flow documents for previewing.
SandcastleBuilder.WPF.PropertyPages
This namespace contains controls and forms used to edit the MSBuild project properties for help file builder projects.
SandcastleBuilder.WPF.Theme
This namespace contains resources and classes used to theme the WPF user controls.
SandcastleBuilder.WPF.UI
This namespace contains user interface related types.
SandcastleBuilder.WPF.UserControls
This namespace contains a set of WPF user controls used by both the standalone GUI and the Visual Studio package to provide a consistent set of features and functionality to the file editors and tool windows used in both.
