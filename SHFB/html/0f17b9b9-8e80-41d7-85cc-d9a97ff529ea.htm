Error SHE0002: Unexpected error: [error]

Cause

This is a generic catch-all error that occurs if a problem is encountered while trying to extract information from the HTML files or produce the output files.

Solution

Since this is a catch-all error, refer to the error description for information about the cause of the error and how you might be able to correct it (i.e. file not found, permission issues, a corrupt file, etc.). If you are unable to solve the problem based on the reported error, open a work item in the Sandcastle Help File Builder project on GitHub containing the details from the error message. Attach the affected project as an example.
