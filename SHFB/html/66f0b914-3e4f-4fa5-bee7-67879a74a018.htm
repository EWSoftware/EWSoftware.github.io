Add Namespace Groups Tool

This tool, AddNamespaceGroups.exe, is used to add namespace group entries to a reflection information file. Namespace groups consist of a list of namespaces that share a common number of root parts. These entries are used to create namespace group topics within the help file similar to the namespace list topics.

See the NamespaceGrouping/MaximumGroupParts project property help for a description of how namespace grouping works.

Command Line Parameters

This tool can be executed from the command line. The following parameters are available.

/out:[outputFile]

Optional. This is used to specify the name of the file to which the updated reflection information is written. If not specified, the output is written to the console.

/maxParts:###

Optional. This is used to specify the maximum number of namespace parts to consider when creating groups. A higher value creates more groups. The default (and minimum) is 2.

reflectionDataFile

Required. Specify the reflection information file to which namespace group information is to be added.

Example Usage
Copy
AddNamespaceGroups.exe /out:reflection.grouped /maxParts:2 reflection.xml
MSBuild Task Properties

This tool can be executed as an MSBuild task. The following properties are available.

WorkingFolder

Optional. This property is used to set the working folder. If not set, the current directory is used as the working folder.

InputFile

Required. This property is used to specify the reflection information file to which namespace group information is added.

OutputFile

Required. This property is used to set the output filename. This file will contain the original reflection data along with any added namespace group entries.

MaximumParts

Optional. This property is used to pass in the maximum number of namespace parts to consider when creating groups. If not specified, the default is 2.

Example Usage
Copy
<UsingTask TaskName="Microsoft.Ddue.Tools.MSBuild.AddNamespaceGroups"
  AssemblyFile="C:\HelpFileBuilderTools\AddNamespaceGroups.exe" />

<PropertyGroup>
  <WorkingFolder></WorkingFolder>
  <MaximumParts>2</MaximumParts>
  <NamespaceGrouping>true</NamespaceGrouping>
</PropertyGroup>

<!-- Execute conditionally based on whether or not namespace grouping is enabled -->
<Microsoft.Ddue.Tools.MSBuild.AddNamespaceGroups
  WorkingFolder="$(WorkingFolder)"
  MaximumParts="$(MaximumParts)"
  InputFile="reflection.ungrouped"
  OutputFile="reflection.xml"
  Condition=" '$(NamespaceGrouping)' == 'true' " />
