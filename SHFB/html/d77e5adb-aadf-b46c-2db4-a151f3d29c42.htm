Microsoft.Ddue.Tools Namespace

This namespace contains the various Sandcastle tools.
Classes
  ClassDescription
AddNamespaceGroupsCore
This utility is used to add namespace groups to a reflection data file. The namespace groups can be used to combine namespaces with a common root into entries in the table of contents in the generated help file.
SegregateByNamespaceCore
This tool takes a reflection data file and splits it into multiple reflection data files, each containing the types from a specific namespace.
XslTransformCore
This tool is used to transform an XML file using one or more specified XSL transformations
