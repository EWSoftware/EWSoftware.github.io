Version 2018.7.8.0

Release notes for version 2018.7.8.0.

Changes in This Release

  • Fixed the syntax generators so that they render ValueType information correctly using the updated reflection data sets.

  • Increased the maximum recursion depth allowed when obtaining nested type info in MRefBuilder (updates a prior fix from 2012).

  • Updated the GenerateInheritedDocs tool to omit the revisionHistory element when inheriting documentation from base types/members.
