Warning GID0009: Circular reference detected: '[member ID stack]'

Cause

This warning occurs if a set of member comments attempts to inherit documentation from itself or a set of parent member comments that have already been encountered thus creating a loop. If this occurs, no further attempt will be made to process inherited comments which may result in incorrect documentation.

Solution

Typically, this is a problem but it may be legitimate if an element inherits comments from another element within the same member comments (i.e. inheriting a span or paragraph from the summary in the remarks element). Review the member ID stack that resulted in the warning. If the circular reference is valid, the warning can be ignored. If the circular reference is not valid, correct the comments to remove the circular reference.
