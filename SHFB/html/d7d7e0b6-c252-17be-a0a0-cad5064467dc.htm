      Sandcastle.Core.Reflection Namespace

      This namespace contains types used to manage reflection data sets.

      Classes

      ApiAssemblyProperties This is used to contain information about an API assembly entry in a reflection information file
      ApiMember This is used to contain information about an API member entry in a reflection information file
      AssemblyDetails This class is used to hold the details for an assembly
      AssemblyLocation This class defines the settings for a specific assembly location
      PlatformType This class holds a set of constants used to define the various .NET platform types
      ReflectionDataSet This class contains information used to obtain reflection data and comments for a specific set of assemblies.
      ReflectionDataSetBindingRedirection This class is used to edit binding redirection settings
      ReflectionDataSetDictionary This dictionary contains reflection data settings for the various .NET Framework platforms and versions

      Enumerations

      ApiMemberGroup This enumerated type represents the groups and subgroups to which an API member can belong
