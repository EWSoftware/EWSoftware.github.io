    MamlTopicRenderer Class

    This is a modified version of the HTML renderer that converts a markdown file to a MAML topic for the Sandcastle Help File Builder build process.

    Remarks

    The result is not a pure MAML document but rather a hybrid of MAML and HTML that can be parsed and rendered to a supported output format by a Sandcastle Help File Builder presentation style.
