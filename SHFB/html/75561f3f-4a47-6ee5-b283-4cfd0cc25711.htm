Remarks
Separate multiple transforms with a semi-colon. Paths starting with "~/" or "~\" are assumed to refer to a Sandcastle transformation file and will be fully qualified with the root Sandcastle tools folder (the folder of the executing assembly). Relative paths are assumed to be relative to the working folder. Absolute paths are not modified.
