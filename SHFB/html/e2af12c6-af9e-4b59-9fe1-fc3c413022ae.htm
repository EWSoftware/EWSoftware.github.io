    BE0037: Could not find the path to the HTML Help 1 compiler

    Error BE0037: Could not find the path to the HTML Help 1 compiler

    Cause

    This error occurs when the HTML Help 1 compiler could not be found in the standard location or anywhere in the Program Files folders on any fixed drive.

    Solution

    The following information can help solve this problem:

    • Review the Installation Instructions carefully and make sure that you have installed all of the required tools and have the latest version of each. The most likely cause is that the HTML Help Workshop is not installed or one or more of its components failed to register correctly. See the referenced installation instructions topic for the download location to install or reinstall it.

    • If all else fails, manually specify the location of the HTML Help 1 compiler by entering the path in the HTML Help 1 Compiler Path project property in the Paths category. This usually needs to be done if performing the build on a build server.
