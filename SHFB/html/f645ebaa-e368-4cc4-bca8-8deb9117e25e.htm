      Error BE0057: The conceptual content file '[file]' with ID '[id]' is not valid: [error]

      Cause

      This error occurs if the given topic file is not valid for some reason (i.e. ill-formed XML, unrecognized entity, etc.).

      Solution

      Refer to the error message for information about the cause of the problem. The most likely reason is ill-formed XML. In that case the error message will contain the line number and column of the element that caused the problem. This will help you locate and fix the problem.
